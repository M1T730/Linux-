# Core Capabilities Introduced

## 1. Data Protection

Backups are introduced to prevent **permanent data loss**.

Typical setup:
- **Proxmox Backup Server**
- scheduled VM/container backups
- retention policies
- restore testing

Example schedule:

| Backup Type | Frequency |
|--------------|-----------|
| VM snapshot | daily |
| container backup | daily |
| configuration backup | weekly |

Testing restore procedures becomes a critical operational practice.

---

## 2. Security Hardening

Basic security controls are implemented to protect services.

Typical protections:
- host firewall rules
- restricted management access
- SSH key authentication
- brute-force protection

Example tools:
- **UFW**
- **Proxmox VE Firewall**
- **Fail2Ban**
- **CrowdSec**

These controls reduce exposure to common attacks.

## 3. Network Segmentation

A **virtualized firewall** introduces proper network boundaries.

Typical platform:
- **pfSense**

Example network structure:
LAN
│
pfSense
├─ VLAN10 → Lab Servers
└─ VLAN20 → Management
Benefits:
- isolates infrastructure from household devices
- protects management interfaces
- introduces enterprise networking concepts
